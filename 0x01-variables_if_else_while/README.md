# 0x01. C - Variables, if, else, while

## Description
This project focuses on control flow statements in C, including if-else constructs and while loops. It also introduces arithmetic, logical, and relational operators, as well as the ASCII character set.

## Requirements
* Language: C (Standard: gnu89)
* OS: Ubuntu 20.04 LTS
* Compiler: gcc -Wall -Werror -Wextra -pedantic -std=gnu89
* Style: WinMingle/Betty Style
