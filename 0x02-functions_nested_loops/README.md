0x02. C - Functions, Nested Loops
Description

This project teaches how to use functions and nested loops in C. learning how to write reusable code and perform repetitive tasks efficiently.

Learning Objectives

Understand functions: declaration, definition, and calling.

Learn nested loops: loops inside loops.

Combine functions and loops to solve problems.

Examples

Printing a multiplication table using nested loops.

Functions to print sequences of numbers.
